# Algerian Forest Fire Prediction

An end-to-end Machine Learning project that predicts Fire Weather Index (FWI) using meteorological data.

## Tech Stack

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy

## Features

- Data preprocessing
- Model training
- Ridge Regression
- Flask web application
- Prediction interface

## Run Locally

pip install -r requirements.txt

python application.py
